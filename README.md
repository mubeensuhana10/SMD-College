# Sri Maata Degree College - Modern Premium Education Platform

A beautifully designed, production-ready React.js website for Sri Maata Degree College (VSKU Affiliated).

## 🚀 Quick Start (Copy & Paste)

```bash
cd SMD-College
node build-project.mjs
npm install
npm run dev
```

Visit `http://localhost:5173` in your browser.

---

## ✨ Key Features

- **Modern UI Design** - Inspired by Tutorly, premium education platform aesthetic
- **Smooth Animations** - Framer Motion with professional transitions
- **Responsive Design** - Mobile-first, optimized for all devices
- **High Conversion** - Strategically placed CTAs and forms
- **SEO Ready** - Meta tags, structured data, sitemap-ready
- **Performance Optimized** - Fast load times, lazy loading, code splitting
- **Component-Based** - Reusable components for easy maintenance

---

## 📋 Homepage Sections

1. **Navbar** - Sticky navigation with transparent scroll effect
2. **Hero Section** - Eye-catching headline with floating stats
3. **Trust Section** - VSKU affiliation, NAAC, badges
4. **About Preview** - 4 feature cards (Faculty, Learning, Skills, Support)
5. **Courses Section** - BCA, BBA, B.Com with career outcomes
6. **Why Choose Us** - 8-item bento grid layout
7. **Campus Life** - Gallery of college activities
8. **Testimonials** - Student reviews carousel
9. **Admission Process** - 4-step timeline
10. **FAQ Section** - Accordion with common questions
11. **Final CTA** - High-impact admissions section
12. **Footer** - Complete with links and social media

---

## 🛠 Tech Stack

- **React 18** - UI Framework
- **Vite 5** - Build Tool
- **Tailwind CSS 3** - Styling
- **Framer Motion 10** - Animations
- **React Router 6** - Navigation
- **Lucide Icons** - Icon Library
